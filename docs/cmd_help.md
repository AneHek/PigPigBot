# /帮助 (help) 指令详细流程

## 概述

返回帮助菜单文本，列出所有可用命令及其说明。纯文本回复，无截图。

注意：`/注册` 命令不在帮助页显示，仅用于兼容旧用户。

## 调用链路

```
用户发送 "/帮助" 或 "/菜单"
    │
    ▼
handler.py :: handle_message()
    │  parse_command("/帮助") → cmd="帮助", arg=""
    │  handlers["帮助"] → game.help()
    │  handlers["菜单"] → game.help()  (别名)
    │
    ▼
pet_game.py :: PetGame.help()
    │
    └─ 直接返回硬编码的帮助文本字符串
       ├─ 基础命令：领养、属性、遗弃、改名、帮助
       ├─ 战斗与成长：战斗 <游戏ID>、进化、训练、休息
       ├─ 排行
       └─ 提示信息
```

## 涉及函数清单

| 函数 | 文件 | 作用 |
|------|------|------|
| `help()` | pet_game.py | 返回帮助文本 |
